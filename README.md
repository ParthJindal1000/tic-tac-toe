# Tic Tac Toe ❌⭕

A clean Python terminal implementation of the classic Tic Tac Toe game. Play against a computer opponent!

---

### Demo

```
==============================
      TIC TAC TOE
==============================
  You are X | Computer is O
  Board positions:
   1 | 2 | 3
  ---|---|---
   4 | 5 | 6
  ---|---|---
   7 | 8 | 9
==============================

   | |
---|---|---
   | |
---|---|---
   | |

Your move (1-9): 5

   | |
---|---|---
   |X|
---|---|---
   | |

Computer is thinking...
Computer placed O at position 1
```

---

### Features

- ❌ Play as X against the computer
- 🤖 Computer plays randomly
- ✅ Detects wins — rows, columns, diagonals
- 🤝 Detects tie games
- 🛡️ Input validation — no crashes on wrong input
- 🧹 Clean board display after every move

---

### Installation

```bash
# Clone the repository
git clone https://github.com/ParthJindal1000/tic-tac-toe

# Run the game (no dependencies needed!)
python tic_tac_toe.py
```

---

### How to Play

- Board positions are numbered 1-9
- Enter a number to place your X
- Computer automatically places O
- First to get 3 in a row wins!
